指南 #

介绍 #

基于 Vue 3 移动端组件库。

特性 #

  • 提供 70+ 套组件，覆盖移动端各类场景
  • 单元测试覆盖
  • 支持按需引入
  • 支持主题定制
  • 支持黑暗模式
  • 支持国际化
  • 支持 TypeScript
  • 支持 SSR

支持环境 #

Android 5.0+

iOS 10.0+

Node.js 12.0+

安装 #

Shell
# npm
npm i -S tantalum-ui-mobile

# yarn
yarn add tantalum-ui-mobile

# pnpm
pnpm add tantalum-ui-mobile
