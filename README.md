# ArkUI

基于 Vue3 移动端组件库。

## 特性

- 提供 70+ 套组件，覆盖移动端各类场景
- 300+ 单元测试覆盖
- 支持按需引入
- 支持主题定制
- 支持黑暗模式
- 支持国际化
- 支持 TypeScript
