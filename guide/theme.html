主题定制 #

组件库的样式是基于 SCSS 开发的，可以通过提供的工具完成主题的个性化定制，主要是色彩相关。

在主题定制之前，推荐通过 色卡 查看下你钟意的颜色在色卡中的效果。推荐颜色值在 HSV 模式下 S > 70% 且 V > 70%。

  1. 打开 主题定制 工具，按操作完成定制并复制定制后的代码。

  2. 新建自定义 SCSS 文件 my-style.scss，粘贴定制代码，如：

    path/to/my-style.scss :

  1. Vite 配置
JavaScript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import Components from 'unplugin-vue-components/vite'
// import { UIResolver } from '@arksjs/ui-resolver'

export default defineConfig({
  plugins: [
    vue(),
    // Components({
    //   dts: true,
    //   resolvers: [UIResolver({ importStyle: 'sass' })]
    // })
  ],
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@use "path/to/my-style.scss" as *;`
      }
    }
  }
})

注：Vue Cli 也有相应的配置。

  1. 安装 scss 依赖

Vite 只需要预装 sass 预处理器，具体请参阅 CSS Pre-processors。

Shell
npm install -D sass

Vue CLI 还需要相应的 loader，具体请参阅 预处理器。

Shell
npm install -D sass-loader sass
